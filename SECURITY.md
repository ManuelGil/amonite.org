# Security Policy

**Scope:** this website repository only.
**Not in scope:** the Amonite GNU/Linux distribution (use that project's channels).

## Reporting

Report website vulnerabilities privately. Do not open public GitHub issues for security reports.

Email: [builder@build.amonite.org](mailto:builder@build.amonite.org)

Include description, reproduction steps, impact, and any suggested fix.

## Expectations

- Acknowledgment within 48 hours
- Status update within 5 business days
- Fix timeline based on severity

Credit is given on fix unless you prefer anonymity.

## Supported

Current production website (`main`) is supported. Older unpublished snapshots are not.

Maintainer: Manuel Gil.
